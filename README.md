# Web Development Portfolio Project

## Description

This repository contains a web development project created through coursework using HTML, CSS, layouts, and database-connected functionality. The project began as a personal portfolio website and expanded into a dynamic web application with multiple pages, user navigation, login/logout features, database-driven content, and interactive profile features.

The application demonstrates front-end structure, page styling, database integration, and user-focused features such as adding, updating, rating, voting, and linking profile information.

## Skills Demonstrated

- HTML structure and semantic markup
- CSS styling and page layout
- Multi-page website development
- Shared layout design and navigation
- About page and profile page creation
- Database integration
- User login and logout functionality
- Adding and updating database records
- Displaying dynamic content from a database
- Voting and rating features
- Linking related profile information
- Web application testing and debugging
- GitHub version control
